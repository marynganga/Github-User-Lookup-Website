Github User Lookup

About

Want to find out information about a Github user? All you have to do is enter their username and click search.

Github User Lookup will do the rest; traversing through Github's database to display the User's Account Info, including:

  • Username
  • Full Name
  • Follower and Following count
  • Github Url
  • Email
  • Avatar

It will also display the User's Public Repositories, detailing the following info for each repository:

  • Name
  • Date Created
  • Description
  • Repository Url
  • Deployed Site
